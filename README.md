## 搬瓦工到底适不适合搭博客？

第三，**套餐梯度覆盖广**。从 $49.99/年 的入门 KVM，到 $169.99/年 的 CN2 GIA-E，再到香港/东京/新加坡的 CN2 GIA 高端机房，预算从一年三百多到一年上万都能找到对应的产品。**对搭博客的人来说，这意味着你能"先用便宜套餐跑起来，有流量再升级"，不用一上来就押宝高端方案**。

## 搭一个博客到底要多少配置？

这个问题其实是整个"搬瓦工适合搭建博客吗"讨论的核心。配置选错了，要么钱白花，要么网站卡到访客直接关页面。

WordPress 官方给出的最低要求是 PHP 7.4、MySQL 5.7、HTTPS 支持。再看常用的 LNMP 和 LAMP 一键环境：装 MySQL 5.7 / MariaDB 10 **要求 1GB 以上内存**，装 PHP 7 及以上**也要求 1GB 以上内存**，LAMP 最低 512MB。也就是说，**1GB 内存是搭博客的现实起步线**，512MB 的方案只能算"勉强能跑"，体验不会好。

按博客规模大致可以这样分：

- **入门级个人博客**：1GB 内存，2 核 CPU，20GB SSD，1TB 流量/月，1Gbps 带宽。日访问量几百到几千 PV 都扛得住。
- **稍具规模的博客**：2GB–4GB 内存，3–4 核 CPU，40GB–80GB SSD，2TB–3TB 流量/月。适合内容多、有图片视频、装了几个插件的中型博客。
- **更高配置**：基本就是有商业化或者高并发的需求了，到这一步你也不需要别人教你怎么选。

硬盘方面，20GB 跑纯文字博客绰绰有余，图片多了建议直接挂对象存储（七牛、又拍、S3 都行），别把图都堆在 VPS 上。流量方面，个人博客 1TB/月几乎用不完，除非你博客突然火了被刷上首页。

## 搬瓦工搭博客的大致流程

如果你看完上面已经动了心，下面这流程大致能让你心里有个数，知道从买到上线要经过几步。

1. **注册账号 + 选套餐下单**。注册不需要翻墙，国内访问搬瓦工镜像站即可。下单时记得在 "Promotional Code" 框里填优惠码，能省一点是一点。👉 [点这里去看当前在售套餐](https://bwh81.net/aff.php?aff=79616&gid=1)
2. **拿到 VPS，登录 KiwiVM**。后台会显示 IP、root 密码、SSH 端口。建议先在 KiwiVM 里重装成你熟悉的系统（Debian 11/12、Ubuntu 22.04 都很常用）。
3. **装建站环境**。两条路：要么用宝塔面板（小白友好，界面化操作），要么用 LNMP/LAMP 一键脚本（轻量、占资源少）。1GB 内存建议 LNMP，宝塔本身要吃掉几百 MB。
4. **绑定域名 + 配置 HTTPS**。域名解析到 VPS IP，用 Let's Encrypt 申请免费 SSL 证书，宝塔和 LNMP 都有一键脚本。
5. **上传 WordPress + 完成安装向导**。WordPress 官网下载 zip，传到网站根目录解压，访问域名按向导填数据库信息即可。
6. **挑主题、装插件、开始写**。到这一步博客已经能访问了，剩下就是内容的事。

整个过程不复杂，从买到上线，一个晚上能搞定。第一次折腾可能慢一点，照着教程一步步来基本不会出大问题。
